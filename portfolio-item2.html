Meditation Mobile Application

UI Design

Nowadays, people are becoming more aware of their mental health because of the growing number of dealing with stress and anxiety. At the same time, it is not always possible for people to go to meditation centers to attend such sessions. The goal of this study is to develop a simple platform for meditation anywhere and anytime.

This project aims to create a visual interface for a mobile meditation application that meets predefined requirements. But for sure, I've surveyed to figure out which features potential users require. My audience was people aged 20-50 years old using some other mediation app. I've chosen five people for an interview from the survey's answers to get deeper insights.

The main goals of this application are to find for users personalized meditation sessions, track users' mood changes, and allow the users to stay stick to a meditation habit.
